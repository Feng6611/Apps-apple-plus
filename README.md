# App Store Region Switcher

[English](./README.md) | [中文](./README_zh.md)

A browser extension that allows you to quickly switch regions while browsing the [apps.apple.com](https://apps.apple.com).

<img src="./public/readme-popup.png" alt="Popup Screenshot" width="300">

## Get Started

Download the extension package from [GitHub Releases](https://github.com/Feng6611/app-store-plus/releases):

- **Chrome**: `app-store-plus-chrome.zip` - For Chrome Web Store
- **Firefox**: `app-store-plus-firefox.zip` - For Firefox Add-ons (AMO)
- **Safari**: `app-store-plus-safari.zip` - For Safari App Extensions

## FAQ

**Q: How do I switch App Store regions?**  
A: After installing the extension, visit any apps.apple.com page, click the extension icon, and select your target region (US, Japan, Hong Kong, etc.) to switch instantly.

## Developer

- Install dependencies: `npm install`
- Development mode: `npm run dev`
- Build all versions: `npm run build:all`
- Package all versions: `npm run zip:all`
