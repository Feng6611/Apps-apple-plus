# App Store 地区切换

[English](./README.md) | [中文](./README_zh.md)

一个浏览器扩展，让你在浏览 [apps.apple.com](https://apps.apple.com) 时快速切换区域。

<img src="./public/readme-popup.png" alt="Popup Screenshot" width="300">

## Get Started

前往 [GitHub Releases](https://github.com/Feng6611/app-store-plus/releases) 下载扩展安装包：

- **Chrome**：`app-store-plus-chrome.zip` - 适用于 Chrome Web Store
- **Firefox**：`app-store-plus-firefox.zip` - 适用于 Firefox Add-ons (AMO)
- **Safari**：`app-store-plus-safari.zip` - 适用于 Safari App Extensions

## FAQ

**Q: 如何切换 App Store 区域？**  
A: 安装扩展后，访问 apps.apple.com 任意页面，点击扩展图标即可选择目标区域（美区、日区、港区等）快速切换。

## Developer

- 安装依赖：`npm install`
- 开发模式：`npm run dev`
- 构建所有版本：`npm run build:all`
- 打包所有版本：`npm run zip:all`
